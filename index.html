CSCI 1300: Responsive Redesign

Redesigning National Collegiate Table Tennis Association Page

Part 1: Identifying Usability Problems

    Why NCTTA website?

    I am a member of the table tennis team. From finding tournament information to reading about the qualification rules, I chose to redesign the National Collegiate Table Tennis Association (NCTTA) website because of how difficult I find using the current website.

    Usability Issues

  1. Lack of headers make the page difficult to follow and take longer to understand. I had a difficult time understanding that below the picture was articles.
  2. The unevenly distributed white space makes the page feel like it is missing something.
  3. The Site Map button is not a poorly named button; I am not sure what that button does.
  4. Recent articles button seems irrelevant, as recent articles are on the page below.
  5. The website has a 1 -> 2 -> ... at bottom of page; unsure of its use.
  6. The news layout is unclear due to lack of identifying information.

    Accessibility Issues

    Expectedly, the website performed poorly on the WAIVE assessment. The website does not have any alt tags specified for its images. This makes the website inaccessible to vision-imparied users. Moreover, the website has a lot of broken links as well.

Part 2: Visual Redesign

Lofi and Hifi Prototypes

Annotated Hifi Prototypes + UI Design

As shown below, I plan on primarily organizing my page through flexbox, as flex box makes it easy to move when the device size changes. The most difficult part will be aligning the picture within the flexbox with my other flexboxes. I plan on using media queries based on width to determine when I should change the size of my text, shift the news side onto the vertical axis, and when to remove the picture. I have it set where my media query sizes based on with are at 1000 and 600 px. I will replace the naviation bar with a hamburger bar at 1000 px for responsiveness with phones and tablets. Moreover, I plan on using the z-index css property to overlay the text onto the front screen image. In order to incorporate the background, I plan on using a linear gradient that is rotated. The font size will be specified almost entirely with em. To make the site more interactive, I will add zoom on hover elements to the news articles as well as the navigation bar items. Finally, for the footer, I plan on using bootstrap to create the table structure.
